วิธีเตรียมใช้งาน 
  **เปิด Note แล้วนำ code ไปวาง และทำการแก้ไขข้อมูลตามที่ต้องการ
  1. เปิด Chrome หรือ browser อื่นๆ
  2. เปิด Bookmark Bar (Ctrl + Shift + B)
  3. คลิกขวา → Add Page
  4. ชื่อ: AutoFill Plate
  5. ช่อง URL วางโค้ดด้านบนทั้งหมด
  6. กด Save

วิธีใช้งาน
  1.เข้าระบบกรมขนส่งและให้เรียบร้อยจนถึงหน้ากรอกข้อมูล
  2.แล้วคลิกที่ Bookmark ที่เราได้เตรียมไว้ 
  3.สคริปจะทำการกรอกให้
